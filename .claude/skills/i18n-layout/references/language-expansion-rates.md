## Font Size Reduction Guidelines

When translated text exceeds container by X%:

| Overflow % | Action |
|------------|--------|
| < 5% | Reduce fontSize by 1pt |
| 5–15% | Reduce fontSize by 2pt |
| 15–30% | Reduce fontSize by 3pt, check readability |
| > 30% | Reduce fontSize 2pt + add line break |

## Minimum Readable Font Sizes (mobile, 2x display)

| Role | Min Size |
|------|----------|

| Title (H2) | 20px |
| Subtitle | 14px |
